# 微雪ESP32-S3 1.54inch LCD 硬件选购指南

## 1. 主板选型

### 推荐型号：微雪 ESP32-S3 1.54inch LCD Module

**规格参数：**
- **主控芯片**: ESP32-S3 (双核Xtensa LX7, 240MHz)
- **内存**: 512KB SRAM + 8MB PSRAM
- **Flash**: 16MB
- **屏幕**: 1.54inch IPS LCD, 240x240分辨率
- **屏幕驱动**: ST7789V
- **通信接口**: SPI (屏幕) + WiFi/蓝牙
- **USB**: Type-C (烧录/供电)
- **尺寸**: 约50x50mm

**购买链接：**
- 微雪官方淘宝店：https://www.waveshare.com/
- 搜索关键词："微雪 ESP32-S3 1.54inch LCD"
- 预计价格：60-90元人民币

## 2. 配件清单

| 配件 | 数量 | 用途 | 备注 |
|------|------|------|------|
| Type-C USB线 | 1 | 烧录/供电 | 需支持数据传输 |

| 3D外壳 | 1 | 保护/美化 | 后期定制 |

## 3. 引脚定义（SPI屏幕）

```
ESP32-S3 引脚 → LCD 引脚
────────────────────────
GPIO 10 → SCLK (SPI时钟)
GPIO 11 → MOSI (SPI数据)
GPIO 12 → CS (片选)
GPIO 13 → DC (数据/命令)
GPIO 14 → RST (复位)
GPIO 21 → BL (背光)
3.3V → VCC
GND → GND
```

## 5. 注意事项

1. **确认型号**：购买前确认是"1.54inch"版本，非"1.28inch圆屏"
2. **PSRAM版本**：选择带8MB PSRAM的版本（用于动画缓存）
3. **天线**：板载PCB天线即可，无需外接天线
4. **烧录器**：无需额外购买，Type-C直连电脑即可
