## 2. 电源设计

### 2.1 电源拓扑

```
USB-C 5V ─┬─ LDO 3.3V ─┬─ ESP32-S3 (VDD)
           │            ├─ LCD (VCC)
           │            └─ 其他外设
           │
           └─ 旁路电容 (100μF电解 + 100nF陶瓷)
```

### 2.2 电源去耦设计

**ESP32-S3 每个VDD引脚：**
- 100nF (0.1μF) 陶瓷电容，X5R/X7R，0402/0603封装
- 尽量靠近芯片引脚放置（<2mm）
- 10μF 电解/陶瓷电容放置在电源入口附近

**LCD VCC：**
- 100nF 陶瓷电容
- 10μF 电解电容（背光驱动需要）

**LDO输入/输出：**
- 输入：100μF 电解 + 100nF 陶瓷
- 输出：100μF 电解 + 100nF 陶瓷

### 2.3 背光驱动

```
GPIO46 (PWM) ──[R 100Ω]── LCD_BL+
                         LCD_BL─ ── GND
```

- PWM频率：1kHz~10kHz
- 占空比0% = 关闭，100% = 最亮
- 电流限制：背光LED典型20mA，最大30mA

## 3. PCB布局建议

### 3.1 层叠设计

**推荐4层板：**
```
Layer 1 (Top)    : 信号 + 器件
Layer 2 (Inner1) : GND 完整平面
Layer 3 (Inner2) : 电源平面
Layer 4 (Bottom) : 信号 + 器件
```

**2层板简化方案：**
```
Layer 1 (Top)    : 信号 + 器件 + 局部铺铜GND
Layer 2 (Bottom) : GND 铺铜 + 少量信号
```

### 3.2 关键布局规则

1. **晶振区域**：ESP32晶振下方保持GND铜皮完整，禁止走线
2. **天线区域**：PCB天线区域禁止铺铜、禁止放置器件
3. **高速信号**：SPI走线长度匹配，阻抗控制50Ω
4. **电源走线**：主电源线宽≥0.5mm，建议1mm
5. **LCD连接器**：FPC连接器靠近板边，便于安装

### 3.3 参考布局图

```
┌──────────────────────────────────────┐
│  USB-C    [LDO]    [ESP32-S3]        │
│           [电容]     │               │
│                      │    [天线区]   │
│  [FLASH]  [晶振]    │    (净空)     │
│                      │               │
│                      │               │
│         [LCD FPC连接器]              │
│  [BOOT]                            │
│  [LED]                              │
└──────────────────────────────────────┘
```

### 3.4 螺丝孔与固定

- 四角 M2 螺丝孔，直径2.2mm
- LCD固定：双面胶 + 3D打印外壳卡扣
- 预留外壳安装柱位

## 4. 散热设计

### 4.1 热源分析

| 热源 | 典型功耗 | 峰值功耗 |
|------|---------|---------|
| ESP32-S3 | 100mW | 500mW (WiFi TX) |
| LCD背光 | 60mW | 100mW |
| LDO | 80mW | 200mW |
| **总计** | **240mW** | **800mW** |

### 4.2 散热措施

**LDO散热：**
- 使用SOT-23或SOT-223封装
- 铺铜散热焊盘，连接内层GND铜皮
- 输入输出压差控制：5V→3.3V压差1.7V，效率66%

**ESP32散热：**
- QFN底部散热焊盘（EP）必须焊接
- 通过过孔连接内层GND平面
- 推荐9个0.3mm直径过孔

**自然对流：**
- 外壳底部开散热孔（直径2mm，间距5mm）
- 避免完全密封外壳

### 4.3 温度监控

ESP32内置温度传感器，可通过软件读取：

```cpp
// 温度监控代码示例（已在固件中实现）

if (temp > 60.0f) {
    // 降低背光
    ledcWrite(BL_CHANNEL, 128);
}
```

## 5. ESD防护设计

### 5.1 防护等级

目标：接触放电 ±4kV，空气放电 ±8kV（IEC 61000-4-2 Level 3）

### 5.2 USB接口防护

```
USB-C D+ ──[ESD保护二极管]── ESP32 D+
USB-C D- ──[ESD保护二极管]── ESP32 D-
USB-C VBUS ──[TVS 5.0SMDJ5.0A]── GND
```

**推荐器件：**
- TVS二极管：5.0SMDJ5.0A（单向，5V钳位）
- ESD保护：PRTR5V0U2X（USB专用）

### 5.3 按键防护

```
GPIO0 ──[R 10kΩ]── 3.3V    (上拉)
     └──[按键]── GND
     └──[ESD]── GND         (可选)
```

- BOOT按键使用内部上拉，外部可加10kΩ上拉提高抗干扰能力
- ESD保护二极管（PESD5V0S1BA）可选，靠近按键放置

### 5.4 LCD接口防护

- FPC连接器处放置ESD阵列（PRTR5V0U4D）
- SPI信号线串联22Ω电阻抑制反射

### 5.5 天线防护

- 天线匹配网络前加气体放电管（GDT）
- 或使用集成ESD保护的天线连接器

## 6. 接口与连接

### 6.1 USB Type-C

```
VBUS ──[5.1kΩ]── CC1    (标识为设备)
     └──[5.1kΩ]── CC2
D+ ──[R 22Ω]── ESP32 D+
D- ──[R 22Ω]── ESP32 D-
GND ── GND
```

### 6.2 LCD FPC连接器

- 8pin FPC，0.5mm间距
- 信号：MOSI, SCLK, CS, DC, RST, BL, VCC, GND
- 连接器型号：FH12-8S-0.5SH

### 6.3 调试接口

预留2.54mm排针：
```
3.3V ── GND ── TX(GPIO43) ── RX(GPIO44)
```
