# 接线指南与引脚图

> 本文档提供桌面电子宠物项目的完整接线参考，包含所有引脚定义、总线拓扑图、电源分配方案及常见接线错误排查。
> 基于微雪 ESP32-S3 1.54inch LCD 开发板，固件引脚定义参见 `esp32_firmware/include/config.h`。

## 一、完整接线表格

### 1.1 SPI 总线 -- LCD 显示屏

| 组件 | ESP32 引脚 | GPIO 编号 | 线色建议 | 方向 | 说明 |
|------|-----------|----------|---------|------|------|
| LCD CS | GPIO5 | 5 | 黄色 | OUT | SPI 片选信号，低电平有效 |

                 │            │                  │            │
                 │   GPIO5  ──┼─── CS   (黄) ───►│ CS         │
                 │            │                  │            │
                 │   GPIO2  ──┼─── DC   (橙) ───►│ DC         │

### 2.2 SPI 信号时序

```
           ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐   ┌───┐
SCLK ──────┘   └───┘   └───┘   └───┘   └───┘   └───┘   └───┘   └───┘   └───
           ┌───────────────────────────────────────────────────────────────────
MOSI ──────┤ D7 │ D6 │ D5 │ D4 │ D3 │ D2 │ D1 │ D0 │
           └───────────────────────────────────────────────────────────────────
             │                                                │
CS   ────────┘                                                └────────────────
             │◄─────────── 8-bit 数据传输 ──────────────────►│

DC=0: MOSI 传输的是命令字节
DC=1: MOSI 传输的是像素数据
```

### 2.3 SPI 总线参数

| 参数 | 值 | 说明 |
|------|---|------|
| SPI 主机 | SPI2_HOST | ESP32-S3 的 SPI2 外设 |
| SPI 模式 | Mode 0 | CPOL=0, CPHA=0 |
| 写入频率 | 80 MHz | LCD 写入速率 |
| 读取频率 | 16 MHz | LCD 读回速率 |
| DMA 通道 | AUTO | 自动分配 DMA 通道 |
| 数据位宽 | 8-bit | 标准 SPI 字节传输 |

### 2.4 SPI 走线注意事项

1. **走线长度**: MOSI、SCLK、CS 走线长度应尽量匹配，误差不超过 5mm
2. **阻抗控制**: SPI 信号线建议控制 50Ω 阻抗
3. **串联电阻**: LCD FPC 连接器处可串联 22Ω 电阻抑制信号反射
4. **远离干扰源**: SPI 走线应远离天线区域和电源走线
5. **地平面**: SPI 走线下方保持完整 GND 铜皮

## 三、I2C 总线接线图

### 3.1 I2C 共享总线拓扑

BH1750 光照传感器和 DRV2605L 触觉驱动器共享同一条 I2C 总线，通过不同的设备地址区分。

```
                         I2C 共享总线 (SDA=GPIO41, SCL=GPIO42)
    ═══════════════════════════════════════════════════════════

    ESP32-S3                    BH1750 模块         DRV2605L 模块
    ┌──────────┐               ┌──────────┐        ┌──────────┐
    │          │               │          │        │          │
    │ GPIO41 ──┼── SDA (蓝) ──►│ SDA      │        │ SDA      │
    │          │         │     │          │        │          │
    │ GPIO42 ──┼── SCL (黄) ──►│ SCL      │        │ SCL      │
    │          │         │     │          │        │          │
    └──────────┘         │     │ 地址:0x23│        │ 地址:0x5A│
                         │     └──────────┘        └──────────┘
                         │
                         ├── 上拉电阻 (4.7kΩ) ──── 3.3V
                         │
                         └── 上拉电阻 (4.7kΩ) ──── 3.3V
```

### 3.2 上拉电阻接线详情

```
    3.3V
     │
     ├───[R 4.7kΩ]─── SDA 总线 (GPIO41)
     │                    │          │
     │                BH1750 SDA   DRV2605L SDA
     │
     └───[R 4.7kΩ]─── SCL 总线 (GPIO42)
                          │          │
                      BH1750 SCL   DRV2605L SCL
```

**上拉电阻选择指南：**

| 总线速率 | 推荐阻值 | 总线电容限制 | 上升时间要求 |
|---------|---------|------------|------------|
| 100kHz (Standard) | 4.7kΩ | < 400pF | < 1000ns |
| 400kHz (Fast) | 2.2kΩ ~ 4.7kΩ | < 400pF | < 300ns |
| 1MHz (Fast-Plus) | 1.0kΩ | < 550pF | < 120ns |

> **注意**: 大多数 BH1750 和 DRV2605L 模块板上已集成上拉电阻 (4.7kΩ)。
> 若多个模块并联，等效阻值会降低。建议确认模块是否带板载上拉，
> 必要时移除板载上拉以避免阻值过低导致功耗增加。

### 3.3 I2C 设备地址表

| 设备 | 7-bit 地址 | 8-bit 写地址 | 8-bit 读地址 | ADDR 配置 |
|------|----------|------------|------------|----------|
| BH1750 | 0x23 | 0x46 | 0x47 | ADDR 接 GND |
| BH1750 (备选) | 0x5C | 0xB8 | 0xB9 | ADDR 接 VCC |
| DRV2605L | 0x5A | 0xB4 | 0xB5 | 固定地址 |

> 本项目中 BH1750 (0x23) 与 DRV2605L (0x5A) 地址完全不同，无冲突。

### 3.4 I2C 通信时序示意

```
时间 ──────────────────────────────────────────────────────────────►

ESP32 (主机)
  │── 读取 BH1750 ──│── 完成 ──│── 间隔 2s ──│── 读取 BH1750 ──│
  │  (120~180ms)    │         │             │  (120~180ms)    │
  │                  │         │             │                  │
  │                  │── 触发 DRV2605L ──│   │                  │
  │                  │  (几ms)          │   │                  │

说明: BH1750 每 2 秒读取一次 (BH1750_READ_INTERVAL=2000ms)
     DRV2605L 为触发式调用（触摸事件时触发）
     两者不会同时占用总线
```

| 电源轨 | 电压 | 供电对象 | 典型电流 | 峰值电流 |
|--------|------|---------|---------|---------|
| USB VBUS | 5.0V | LDO 输入、背光 LED | 150mA | 500mA |

### 4.3 去耦电容接线

```
LDO 输入端 (5V):
  5V ──┬──[100μF 电解]── GND
       └──[100nF 陶瓷]── GND

LDO 输出端 (3.3V):
  3.3V ──┬──[100μF 电解]── GND
         └──[100nF 陶瓷]── GND

ESP32-S3 VDD 引脚:
  每个 VDD ──[100nF 陶瓷 (X5R, 0402)]── GND
  (尽量靠近芯片引脚放置，距离 < 2mm)

LCD VCC:
  3.3V ──┬──[10μF]── GND
         └──[100nF]── GND

BH1750 VCC:
  3.3V ──[100nF]── GND

DRV2605L VDD:
  3.3V ──[100nF]── GND
```

### 4.4 背光驱动电路

```
GPIO48 ──[R 100Ω]──► LCD BL+
                       LCD BL- ──► GND

计算: I = (5V - 3.2V) / 100Ω ≈ 18mA (安全范围内)
```

## 六、常见接线错误与排查

### 6.1 LCD 无显示

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 完全黑屏 | 背光未接或 BL 引脚错误 | 用万用表测量 BL+ 是否有电压 (PWM 输出) |
| 白屏 | MOSI/SCLK 接线错误或松动 | 检查 SPI 信号线是否正确连接 |
| 颜色异常 | DC 引脚接反 | 确认 DC 引脚连接，高=数据，低=命令 |
| 显示偏移 | CS 或 RST 时序问题 | 检查 CS 和 RST 接线是否牢固 |
| 闪烁 | 电源不稳或去耦不足 | 检查 3.3V 供电，增加去耦电容 |

### 6.2 I2C 设备无法通信

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 扫描不到设备 | SDA/SCL 接反 | 交换 SDA 和 SCL 接线 |
| 扫描不到设备 | 缺少上拉电阻 | 确认模块是否带板载上拉，必要时外接 4.7kΩ |
| 地址冲突 | ADDR 引脚配置错误 | 用 I2C Scanner 扫描确认地址 |
| 读数异常 | 总线速率过高 | 降低 I2C 速率至 100kHz 测试 |
| 偶发失败 | 总线电容过大 | 减少总线分支长度，或降低上拉电阻值 |

**I2C Scanner 代码片段 (Arduino):**
```cpp
#include <Wire.h>
Wire.begin(41, 42);  // SDA=GPIO41, SCL=GPIO42
for (byte addr = 1; addr < 127; addr++) {
    Wire.beginTransmission(addr);
    if (Wire.endTransmission() == 0) {
        Serial.printf("发现设备: 0x%02X\n", addr);
    }
}
```

### 6.3 蜂鸣器不响或音调异常

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 完全无声 | 使用了有源蜂鸣器 | 确认是**无源**蜂鸣器（有源蜂鸣器无法控制音调） |
| 完全无声 | GPIO18 接线错误 | 检查正负极是否接反 |
| 声音微弱 | GPIO 驱动能力不足 | 增加 NPN 三极管 (S8050) 放大驱动 |
| 音调不准 | PWM 频率设置有误 | 检查固件中的频率配置 |

### 6.4 触摸无响应

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 无触摸反应 | GPIO1 未连接触摸铜箔 | 确认触摸引脚连接到导电面 |
| 灵敏度过低 | 阈值设置过大 | 降低 `TOUCH_THRESHOLD` (当前默认 40) |
| 误触发 | 阈值设置过小 | 增大 `TOUCH_THRESHOLD` |
| 不稳定 | 走线过长产生寄生电容 | 缩短触摸引脚走线，远离电源线 |

### 6.5 WiFi 连接问题

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| 搜不到 WiFi | 天线区域被遮挡 | 确认 PCB 天线区域无金属遮挡 |
| 信号弱 | 天线附近有走线或铺铜 | 检查天线净空区 (参考 PCB 布局规范) |
| 频繁断连 | 电源不稳导致射频异常 | 检查 3.3V 供电稳定性，增加去耦电容 |
| 连接慢 | DTIM 间隔过大 | 调整 `DTIM_IDLE` 参数 |

### 6.6 通用排查工具

| 工具 | 用途 | 推荐型号 |
|------|------|---------|
| 万用表 | 测量电压、通断 | 任意数字万用表 |
| 逻辑分析仪 | 检查 SPI/I2C 波形 | Saleae Logic / Kingst LA1010 |
| 串口监视器 | 查看固件调试输出 | Arduino Serial Monitor / PuTTY |
| I2C Scanner | 扫描总线设备地址 | 固件内置或 Arduino 示例 |
| USB 电流表 | 监测系统功耗 | Type-C 电流表 |

## 七、不同开发板的引脚差异

如果使用非微雪官方开发板，引脚映射可能不同，需要修改 `config.h`。

### 7.1 常见开发板引脚对照

| 功能 | 微雪 (本项目) | 安信可 ESP32-S3 | 乐鑫 ESP32-S3-DevKitC |
|------|-------------|---------------|---------------------|
| LCD CS | GPIO5 | GPIO10 | GPIO10 |
| LCD DC | GPIO2 | GPIO13 | GPIO9 |
| LCD RST | GPIO4 | GPIO14 | GPIO46 |
| LCD MOSI | GPIO11 | GPIO6 | GPIO11 |
| LCD SCLK | GPIO12 | GPIO7 | GPIO12 |
| LCD BL | GPIO48 | GPIO21 | GPIO48 |

### 7.2 修改引脚配置

修改 `esp32_firmware/include/config.h` 中的宏定义：

```cpp
// 根据实际开发板修改以下引脚
#define LCD_CS    5       // 改为实际 CS 引脚
#define LCD_RST   4       // 改为实际 RST 引脚
#define LCD_DC    2       // 改为实际 DC 引脚
#define LCD_MOSI  11      // 改为实际 MOSI 引脚
#define LCD_SCLK  12      // 改为实际 SCLK 引脚
#define LCD_BL    48      // 改为实际 BL 引脚
#define LCD_MISO  13      // 改为实际 MISO 引脚
```
