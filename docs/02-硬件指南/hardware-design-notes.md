# 硬件设计参考

> 本文档为桌面电子宠物项目的硬件设计参考，涵盖电源设计、PCB 布局、热管理、ESD 防护、天线设计、测试点定义和 BOM 成本分析。
> 基于微雪 ESP32-S3 1.54inch LCD 开发板平台，适用于自行设计 PCB 的量产阶段。
> 非专业硬件工程师编写，仅供参考，实际生产需由专业人员评审。

## 一、电源设计

### 1.1 电源拓扑

```
USB-C 5V ─┬─ LDO 3.3V ─┬─ ESP32-S3 (VDD, 3.3V)
           │            ├─ ST7789V LCD (VCC, 3.3V)
           │            ├─ BH1750 (VCC, 3.3V)
           │            └─ DRV2605L (VDD, 3.3V)
           │
           └─ LCD 背光 LED (5V 直连，经限流电阻)
```

### 1.2 LDO 选型

| 参数 | AMS1117-3.3 | RT9080-33GJ5 | ME6211C33 | 推荐 |
|------|------------|-------------|-----------|------|
| 输出电压 | 3.3V | 3.3V | 3.3V | - |
| 最大输出电流 | 800mA | 500mA | 500mA | - |
| 压差 (Dropout) | 1.1V | 0.25V | 0.15V | RT9080 |
| 静态电流 | 5mA | 50μA | 40μA | ME6211 |
| 封装 | SOT-223 / SOT-23 | SOT-23-5 | SOT-23-5 | - |
| 价格 | ~0.3 元 | ~0.5 元 | ~0.4 元 | - |
| 本项目适用性 | 原型验证 | **推荐量产** | 低功耗方案 | RT9080 |

**LDO 选型建议**:
- **原型阶段**: 使用 AMS1117-3.3 (开发板自带)，成本低，货源充足
- **量产阶段**: 推荐 RT9080-33GJ5，低压差 (0.25V) 减少发热，静态电流低
- **低功耗方案**: ME6211C33，40μA 静态电流适合电池供电场景

### 1.3 去耦电容设计

**ESP32-S3 每个 VDD 引脚**:
- 100nF (0.1μF) 陶瓷电容，X5R/X7R，0402/0603 封装
- 尽量靠近芯片引脚放置 (距离 < 2mm)
- 10μF 电解/陶瓷电容放置在电源入口附近

**LCD VCC**:
- 100nF 陶瓷电容
- 10μF 电解电容 (背光驱动需要)

**LDO 输入/输出**:

| 位置 | 电容值 | 类型 | 封装 | 说明 |
|------|--------|------|------|------|
| LDO 输入 | 100μF | 电解 | 6.3V | 输入滤波 |
| LDO 输入 | 100nF | 陶瓷 X5R | 0402 | 高频去耦 |
| LDO 输出 | 100μF | 电解 | 6.3V | 输出滤波 |
| LDO 输出 | 100nF | 陶瓷 X5R | 0402 | 高频去耦 |
| ESP32 VDD (每个) | 100nF | 陶瓷 X5R | 0402 | 芯片级去耦 |
| LCD VCC | 10μF + 100nF | 陶瓷 | 0805/0402 | LCD 供电 |
| BH1750 VCC | 100nF | 陶瓷 X5R | 0402 | 传感器去耦 |
| DRV2605L VDD | 100nF | 陶瓷 X5R | 0402 | 驱动器去耦 |

### 1.4 功耗预算

| 模块 | 典型功耗 | 峰值功耗 | 典型电流 @3.3V |
|------|---------|---------|---------------|
| ESP32-S3 (WiFi 活跃) | 330mW | 1650mW | 100~500mA |
| ESP32-S3 (Light Sleep) | 16.5mW | 33mW | 5~10mA |
| LCD 背光 | 60mW | 100mW | 18~30mA |
| ST7789V 驱动 | 15mW | 30mW | 5~10mA |
| BH1750 | 0.4mW | 0.6mW | 0.12~0.19mA |
| DRV2605L (待机) | 0.15mW | 0.3mW | 0.05~0.1mA |
| DRV2605L (驱动 LRA) | 16.5mW | 33mW | 5~10mA |
| 蜂鸣器 | 33mW | 66mW | 10~20mA |
| **总计 (WiFi 活跃)** | **~455mW** | **~1880mW** | **~138~570mA** |
| **总计 (Light Sleep)** | **~145mW** | **~233mW** | **~44~70mA** |

**LDO 功耗计算 (5V → 3.3V)**:
```
P_LDO = (V_in - V_out) × I_load
      = (5.0 - 3.3) × 0.4A (典型负载)
      = 0.68W (典型)
      = (5.0 - 3.3) × 0.57A (峰值负载)
      = 0.97W (峰值)
```

> LDO 在峰值负载时功耗接近 1W，需注意散热设计 (见第三节)。

### 1.5 电源时序

```
时间 ──────────────────────────────────────────────────────►

```

## 二、PCB 布局建议

### 2.1 层叠设计

**推荐 4 层板 (量产)**:

```
Layer 1 (Top)     : 信号 + 器件
Layer 2 (Inner 1) : GND 完整平面
Layer 3 (Inner 2) : 电源平面 (3.3V)
Layer 4 (Bottom)  : 信号 + 器件
```

**2 层板简化方案 (原型)**:

```
Layer 1 (Top)     : 信号 + 器件 + 局部铺铜 GND
Layer 2 (Bottom)  : GND 铺铜 + 少量信号
```

| 层叠方案 | 成本 | 信号完整性 | EMI 性能 | 适用阶段 |
|---------|------|----------|---------|---------|
| 2 层板 | 低 | 一般 | 一般 | 原型验证 |
| 4 层板 | 中 | 良好 | 良好 | 小批量试产 |
| 4 层板 (阻抗控制) | 中高 | 优秀 | 优秀 | 量产 |

### 2.2 SPI 走线规则

LCD 使用 SPI 接口，最高 80MHz，需注意高速信号完整性：

| 规则 | 要求 | 说明 |
|------|------|------|
| 走线长度匹配 | MOSI/SCLK/CS 误差 < 5mm | 避免时序偏移 |
| 阻抗控制 | 50Ω 单端阻抗 | 减少信号反射 |
| 串联电阻 | LCD FPC 处串联 22Ω | 抑制过冲和反射 |
| 走线间距 | ≥ 2 倍线宽 | 减少串扰 |
| 参考平面 | 下方保持完整 GND | 确保回流路径 |
| 过孔数量 | 尽量少，最多 2 个 | 减少阻抗不连续 |

**SPI 走线示意**:
```
                    ┌─────────────────────────────────────┐
                    │            22Ω 串联电阻              │
ESP32 GPIO11 (MOSI) ──[22Ω]──────────────────────────────►│ FPC
ESP32 GPIO12 (SCLK) ──[22Ω]──────────────────────────────►│ 连接器
ESP32 GPIO5  (CS)   ──[22Ω]──────────────────────────────►│
ESP32 GPIO2  (DC)   ─────────────────────────────────────►│
ESP32 GPIO4  (RST)  ─────────────────────────────────────►│
                    └─────────────────────────────────────┘
```

### 2.3 I2C 走线规则

| 规则 | 要求 | 说明 |
|------|------|------|
| 上拉电阻位置 | 靠近主机端 (ESP32) | 减少总线电容影响 |
| 走线长度 | < 30cm (400kHz) | 超长走线需降低速率 |
| 走线间距 | SDA/SCL 平行走线 < 5cm | 减少串扰 |
| 上拉电阻值 | 4.7kΩ (标准) / 2.2kΩ (高速) | 根据总线速率选择 |
| 总线电容 | < 400pF | 含走线 + 器件寄生电容 |

**I2C 上拉电阻布局**:
```
                    3.3V
                     │
                     ├──[4.7kΩ]── SDA (GPIO41)
                     │               │          │
                     │           BH1750 SDA   DRV2605L SDA
                     │
                     └──[4.7kΩ]── SCL (GPIO42)
                                     │          │
                                 BH1750 SCL   DRV2605L SCL

布局要求:
  - 上拉电阻尽量靠近 GPIO41/GPIO42 放置
  - SDA/SCL 走线尽量短 (< 10cm)
  - 避免与高速信号线平行走线
```

### 2.4 EMI 抑制措施

| 措施 | 位置 | 说明 |
|------|------|------|
| GND 铺铜 | 全板 | 提供低阻抗回流路径 |
| 电源平面 | 内层 | 减少电源环路面积 |
| 去耦电容 | 每个 IC 电源引脚旁 | 抑制高频噪声 |
| 串联电阻 | SPI 信号线 | 抑制过冲和振铃 |
| 地过孔阵列 | 信号换层处 | 确保回流路径连续 |
| 铁氧体磁珠 | 电源入口 | 隔离 USB 噪声 |

### 2.5 参考布局图

```
┌──────────────────────────────────────────────────────────────┐
│  USB-C         [LDO]         [ESP32-S3]                      │
│  连接器        [电容组]        ┌─────────┐                    │
│                              │  QFN56  │    [PCB 天线区]     │
│                              │  7x7mm  │     (净空区)         │

│  8MB           40MHz         └─────────┘     禁止走线         │

│  [BH1750]      [DRV2605L]                   天线净空 ≥5mm    │

螺丝孔: 四角 M2, 直径 2.2mm

```

### 2.6 螺丝孔与固定

- 四角 M2 螺丝孔，直径 2.2mm
- LCD 固定: 双面胶 + 3D 打印外壳卡扣
- 预留外壳安装柱位
- 螺丝孔周围 2mm 内禁止走线和铺铜

## 三、热管理

### 3.1 热源分析

| 热源 | 封装 | 典型功耗 | 峰值功耗 | 最高结温 |
|------|------|---------|---------|---------|
| ESP32-S3 | QFN56 (7x7mm) | 100mW | 500mW (WiFi TX) | 125°C |
| LCD 背光 LED | FPC 模块 | 60mW | 100mW | 85°C |
| AMS1117 LDO | SOT-223 | 80mW | 970mW | 125°C |
| **总计** | - | **240mW** | **~1570mW** | - |

### 3.2 ESP32-S3 散热设计

**QFN 底部散热焊盘 (EP)**:
- 必须焊接散热焊盘
- 通过过孔连接内层 GND 平面
- 推荐 9 个 0.3mm 直径过孔，阵列排列

```
ESP32-S3 底部过孔阵列:

    ○  ○  ○       ○ = 0.3mm 过孔
    ○  ○  ○       间距: 1.2mm
    ○  ○  ○       连接: Top → Inner1 (GND)
```

**散热铜皮**:
- EP 焊盘周围铺铜面积 ≥ 10mm x 10mm
- 连接到内层 GND 平面 (4 层板) 或底层铺铜 (2 层板)
- 避免在散热区域走信号线

### 3.3 LDO 散热设计

**AMS1117 (SOT-223) 散热**:
- Tab 引脚连接到大面积铺铜
- 输入输出压差: 5V → 3.3V = 1.7V
- 效率: 3.3/5.0 = 66%
- 峰值功耗: 1.7V × 0.57A ≈ 0.97W

**散热改进方案**:
1. 使用低压差 LDO (RT9080, 压差 0.25V)
   - 功耗: 0.25V × 0.57A ≈ 0.14W (降低 85%)
2. 使用 DC-DC 降压模块
   - 效率 > 90%，几乎无发热
   - 缺点: 增加成本和 EMI

### 3.4 LCD 背光散热

```
背光 LED 参数:
  典型电流: 20mA
  最大电流: 30mA
  正向压降: ~3.2V
  功耗: 3.2V × 20mA = 64mW (典型)

散热措施:
  - 背光 LED 通过 FPC 连接器散热
  - FPC 连接器附近铺铜辅助散热
  - PWM 调光减少平均功耗
```

### 3.5 自然对流设计

| 措施 | 位置 | 说明 |
|------|------|------|
| 散热孔 | 外壳底部 | 直径 2mm，间距 5mm |
| 通风槽 | 外壳侧面 | 宽 1mm，长 10mm |
| 避免密封 | 外壳整体 | 留有空气流通缝隙 |
| 脚垫高度 | 外壳底部 | ≥ 3mm，增加底部通风空间 |

### 3.6 温度监控

ESP32 内置温度传感器，固件可实时监控：

```cpp
// 温度监控代码 (已在固件中实现)
float temp = temperatureRead();
if (temp > 60.0f) {
    // 降低背光亮度，减少发热
    ledcWrite(BL_CHANNEL, 128);
}
if (temp > 80.0f) {
    // 进入低功耗模式
    WiFi.setSleep(true);
}
```

**温度阈值建议**:

| 温度 | 状态 | 动作 |
|------|------|------|
| < 50°C | 正常 | 全速运行 |
| 50~60°C | 偏高 | 降低背光亮度 |
| 60~80°C | 过热 | 关闭 WiFi，降低 CPU 频率 |
| > 80°C | 危险 | 进入休眠模式，告警 |

## 四、ESD 防护设计

### 4.1 防护等级目标

| 标准 | 等级 | 接触放电 | 空气放电 |
|------|------|---------|---------|
| IEC 61000-4-2 | Level 3 | ±4kV | ±8kV |

### 4.2 USB 接口防护

```
USB-C D+ ──[PRTR5V0U2X]── ESP32 D+
USB-C D- ──[PRTR5V0U2X]── ESP32 D-
USB-C VBUS ──[5.0SMDJ5.0A]── GND
```

| 器件 | 型号 | 封装 | 功能 | 价格 |
|------|------|------|------|------|
| TVS 二极管 | 5.0SMDJ5.0A | SMA | VBUS 过压保护 | ~0.5 元 |
| ESD 保护 | PRTR5V0U2X | SOT-23-6 | USB 数据线 ESD | ~0.3 元 |
| ESD 保护 (备选) | USBLC6-2SC6 | SOT-23-6 | USB 专用 ESD | ~0.4 元 |

### 4.3 按键防护

```
GPIO0 ──[R 10kΩ]── 3.3V    (外部上拉)
     └──[按键]── GND
     └──[PESD5V0S1BA]── GND (ESD 保护, 可选)
```

- BOOT 按键使用内部上拉 (~45kΩ)，外部加 10kΩ 上拉提高抗干扰
- ESD 保护二极管 PESD5V0S1BA (SOD-323) 可选，靠近按键放置

### 4.4 LCD 接口防护

- FPC 连接器处放置 ESD 阵列 PRTR5V0U4D (SOT-23-6)
- SPI 信号线串联 22Ω 电阻抑制反射 (兼做限流)
- VCC 入口加 100nF 去耦 + TVS 保护

### 4.5 I2C 总线防护

```
GPIO41 (SDA) ──[PESD5V0S1BA]── GND
GPIO42 (SCL) ──[PESD5V0S1BA]── GND
```

- I2C 引脚暴露在外，需加 ESD 保护
- PESD5V0S1BA 靠近 ESP32 端放置
- 上拉电阻放在 ESD 保护之后 (靠近总线端)

### 4.6 天线防护

- 天线匹配网络前加气体放电管 (GDT)
- 或使用集成 ESD 保护的天线连接器
- PCB 天线无需额外防护 (已集成在芯片内部)

## 五、天线设计

### 5.1 PCB 天线净空区要求

ESP32-S3-WROOM-1 使用板载 PCB 天线，需确保天线区域无遮挡：

```
┌──────────────────────────────────────────┐
│                                          │
│  ┌──────────────┐   ┌──────────────────┐ │
│  │              │   │                  │ │
│  │   ESP32-S3   │   │  天线净空区      │ │
│  │   模组区域    │   │                  │ │
│  │              │   │  禁止:           │ │
│  │              │   │  - 铺铜          │ │
│  └──────────────┘   │  - 走线          │ │
│                     │  - 放置器件      │ │
│                     │  - 金属外壳      │ │
│                     │                  │ │
│                     │  净空距离:       │ │
│                     │  ≥ 5mm (推荐)    │ │
│                     │                  │ │
│                     └──────────────────┘ │
│                                          │
└──────────────────────────────────────────┘
```

### 5.2 天线性能影响因素

| 因素 | 影响 | 解决方案 |
|------|------|---------|
| 金属外壳 | 严重衰减信号 | 使用塑料外壳或开天线窗 |
| 金属底座 | 反射信号 | 底座留空或使用非金属材料 |
| 电池遮挡 | 衰减信号 | 电池远离天线区域 |
| 人体遮挡 | 衰减信号 | 天线朝向外侧 |
| PCB 铺铜 | 改变天线阻抗 | 严格遵守净空区要求 |

### 5.3 外置天线选项

若 PCB 天线信号不足，可使用 IPEX 连接器外接天线：

| 天线类型 | 增益 | 尺寸 | 价格 | 适用场景 |
|---------|------|------|------|---------|
| PCB 板载天线 | ~2dBi | 集成 | 0 元 | 一般场景 |
| IPEX 陶瓷天线 | ~3dBi | 5x3mm | 2 元 | 紧凑设计 |
| IPEX 弹簧天线 | ~3dBi | 10mm | 1 元 | 经济方案 |
| SMA 外置天线 | ~5dBi | 10cm | 5 元 | 远距离需求 |

> 本项目使用板载 PCB 天线即可满足桌面场景需求。
