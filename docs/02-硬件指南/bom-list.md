## 二、核心组件详情

### 2.1 ESP32-S3 LCD 开发板

**推荐型号**: 微雪 ESP32-S3 1.54inch LCD Module

| 参数 | 规格 |
|------|------|
| 主控 | ESP32-S3FN8 (双核 LX7, 240MHz) |
| 内存 | 512KB SRAM + 8MB PSRAM (OPI) |
| Flash | 16MB (Quad SPI) |
| 屏幕 | 1.54" IPS LCD, 240x240, ST7789V |
| 接口 | SPI (LCD) + USB-C + WiFi + BLE 5.0 |
| 尺寸 | 约 50mm x 50mm |
| 天线 | 板载 PCB 天线 |

**选型要点**:
- 必须选择带 **8MB PSRAM** 的版本 (动画帧缓冲必需)
- 确认是 **1.54inch 方屏** 版本，非 1.28inch 圆屏
- 微雪官方品质稳定，引脚定义与固件一致
- 安信可等替代品牌引脚可能不同，需修改 `config.h`

**替代型号对比**:

| 型号 | PSRAM | 屏幕 | 价格 | 引脚兼容 |
|------|-------|------|------|---------|
| 微雪 ESP32-S3 1.54"LCD | 8MB OPI | 240x240 IPS | 70 元 | 原生支持 |
| 安信可 ESP32-S3-1.54" | 8MB OPI | 240x240 IPS | 60 元 | 需修改引脚 |
| 乐鑫 DevKitC-1 + 外接 LCD | 8MB OPI | 需另购 | 45+30 元 | 需自行接线 |

### 2.2 BH1750 光照传感器模块

**推荐型号**: GY-302 (BH1750FVI 模块)

| 参数 | 规格 |
|------|------|
| 芯片 | BH1750FVI (ROHM) |
| I2C 地址 | 0x23 (ADDR 接 GND) |
| 测量范围 | 1 ~ 65535 lux |
| 分辨率 | 1 lux (高分辨率模式) |
| 精度 | ±20% (典型) |
| 工作电压 | 2.4V ~ 3.6V |
| 工作电流 | 120μA (测量), 0.01μA (断电) |
| 模块特性 | 板载上拉电阻、稳压电路 |

**选型要点**:
- 选择已焊接排针的版本，减少手工焊接
- 确认 ADDR 引脚默认接 GND (地址 0x23)
- 模块板载 4.7kΩ 上拉电阻，无需外接
- 价格低廉 (3~5 元)，建议多买 1~2 个备用

### 2.3 DRV2605L 触觉驱动模块

**推荐型号**: GY-DRV2605L (TI DRV2605L Breakout Board)

| 参数 | 规格 |
|------|------|
| 芯片 | DRV2605L (Texas Instruments) |
| I2C 地址 | 0x5A (固定) |
| 内置效果 | 123 种预设触觉效果 |
| 支持马达 | LRA (线性谐振) + ERM (偏心旋转) |
| 工作电压 | 2.0V ~ 5.5V |
| 输出驱动 | LRA: 2.0Vrms, ERM: 2.5V |
| 模块特性 | 板载上拉电阻、FPC 马达接口 |

**选型要点**:
- 确认模块附带 FPC 连接器或焊接点用于连接 LRA 马达
- 若模块不含 LRA 马达，需单独采购 10mm 圆形 LRA
- 模块板载上拉电阻，与 BH1750 共享总线时注意并联阻值
- 价格 8~15 元，TI 原片品质可靠

### 2.4 LRA 线性马达

| 参数 | 规格 |
|------|------|
| 类型 | LRA (Linear Resonant Actuator) |
| 尺寸 | 10mm 直径圆形 |
| 额定电压 | 1.8Vrms |
| 谐振频率 | 170Hz ± 10Hz |
| 阻抗 | 8Ω ± 2Ω |

**选型要点**:
- 选择谐振频率 150~200Hz 的 LRA，与 DRV2605L Library 6 效果匹配
- 10mm 尺寸振动感明显，8mm 尺寸适合更紧凑的外壳
- 建议采购时确认附带导线，方便焊接

### 2.5 无源蜂鸣器

| 参数 | 规格 |
|------|------|
| 类型 | 无源蜂鸣器 (需外部方波驱动) |
| 工作电压 | 3.3V ~ 5V |
| 频率范围 | 500Hz ~ 10kHz |
| 工作电流 | ~10mA |
| 尺寸 | 12mm 直径 |

**选型要点**:
- **必须选择无源蜂鸣器**，有源蜂鸣器无法控制音调
- 无源蜂鸣器无 "+" "-" 标识，有源蜂鸣器通常有标识
- 3.3V 供电的无源蜂鸣器音量较小，5V 版本音量更大
- 本项目使用 GPIO18 直接驱动，无需外接三极管

## 六、注意事项

### 6.1 采购前确认清单

- [ ] 确认开发板是 **1.54inch 方屏** 版本，非 1.28inch 圆屏
- [ ] 确认开发板带 **8MB PSRAM** (动画缓存必需)
- [ ] 确认 Type-C 线 **支持数据传输**，非纯充电线
- [ ] 确认蜂鸣器是 **无源** 类型 (有源蜂鸣器无法控制音调)
- [ ] 确认 BH1750 模块 ADDR 引脚可配置 (默认接 GND = 0x23)
- [ ] 确认 DRV2605L 模块附带 LRA 马达接口或已含马达

### 6.2 收货后验证步骤

1. **开发板**: USB 连接电脑，确认识别为串口设备
2. **BH1750**: 使用 I2C Scanner 确认地址 0x23
3. **DRV2605L**: 使用 I2C Scanner 确认地址 0x5A
4. **蜂鸣器**: 手动给 GPIO18 输出方波，确认有声音
5. **Type-C 线**: 确认能传输数据 (非仅充电)

### 6.3 常见采购坑

| 问题 | 说明 | 解决方案 |
|------|------|---------|
| 买成圆屏 | 1.28inch 圆屏引脚完全不同 | 仔细核对商品标题 |
| PSRAM 不足 | 部分版本只有 2MB PSRAM | 查看商品详情页规格表 |
| 充电线无数据 | 部分 Type-C 线仅支持充电 | 选择标注"数据传输"的线 |
| 有源蜂鸣器 | 有源蜂鸣器音调固定 | 选择无标识或标注"无源"的蜂鸣器 |
| 模块不带排针 | 部分模块需自行焊接排针 | 选择"已焊接排针"版本 |

### 6.4 备件建议

建议多采购以下易损/易丢组件作为备件：

| 组件 | 建议备件数 | 原因 |
|------|----------|------|
| BH1750 模块 | +1 | 便宜，接线错误可能损坏 |
| 杜邦线 | +5 根 | 接触不良时需更换 |
| LRA 马达 | +1 | 引线易断 |
| 无源蜂鸣器 | +1 | 便宜，焊接可能损坏 |
